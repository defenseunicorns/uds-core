A UDS Package wraps a Helm chart with a UDS Package custom resource (CR) that declares how your application integrates with the platform — networking, SSO, monitoring, and more. The UDS Operator reads the Package CR and configures Istio, Keycloak, Prometheus, and network policies automatically.

These guides walk you through packaging your own application for deployment on UDS Core, from wrapping a Helm chart to publishing the finished artifact.

> [!TIP]
> New to UDS packaging? Start with the [Bundles](/concepts/configuration-and-packaging/bundles/) and [CRD Overviews](/concepts/configuration-and-packaging/crd-overviews/) concept pages to understand how packages, bundles, and the Package CR fit together.
