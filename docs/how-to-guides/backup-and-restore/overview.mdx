UDS Core provides cluster backup and restore through [Velero](https://velero.io/). This section covers configuring storage backends, enabling volume snapshots, and performing backup and restore operations.

For background on how Velero works and what it backs up, see [Backup & restore concepts](/concepts/core-features/backup-restore/).
